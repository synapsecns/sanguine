# $SYN Token

$SYN is the governance token for the Synapse Protocol. SYN holders can participate in governance decisions for [Synapse Protocol](https://synapseprotocol.com/), [Hypercall](https://hypercall.xyz/), and [Cortex Protocol](https://cortexprotocol.com/) through the [Synapse DAO](/docs/About/DAO).

SYN is listed on major exchanges including Binance and Kraken, providing easy access for users worldwide.

All future $SYN emissions are governed by the [DAO](/docs/About/DAO).
